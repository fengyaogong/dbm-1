To make this work please:
Download from http://yann.lecun.com/exdb/mnist the following 4 files:
train-images-idx3-ubyte.gz
train-labels-idx1-ubyte.gz
t10k-images-idx3-ubyte.gz
t10k-labels-idx1-ubyte.gz

Add them to the data directory and unzip them.

Run demo.m in MatLab, but please note that running can take few days 
even on powerfull machines.
